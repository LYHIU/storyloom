# 书织 StoryLoom

> 书织回文锦，无因寄陇头。  
> 墨通山海路，梭停万象全。

**书织以字为经、以章为纬，陪你织起人物、情节与世界。当梭声停下，万象俱全。**

书织是一款为网文写手设计的桌面写作工具。从灵感到完本，四个场景化空间覆盖写作全流程，数据全部本地存储，AI 只辅助不替代。

## 产品概念

"书织"之名源自**数织（Nonogram）**——一种通过数字线索逐格填色的像素游戏。正如数织中每一格都在为最终图案铺路，书织让每一次敲击都成为织就作品的经纬。

每本新书都会随机生成一只独一无二的**像素伙伴（Pixel Buddy）**陪伴你完成创作。你码字，它从像素中渐渐浮现——写够字数点亮一格，从灰暗到彩色，从模糊到清晰。16 种活动场景（进食、打盹、玩耍、奔跑……）串联整本书的旅程，写完最后一字，伙伴完成它全部的陪伴。

## 当前状态

**Phase 1 已完成** — 码字空间可用。

## 功能概览

### Phase 1 ✓
- **码字空间** — 编辑器（tiptap）、大纲侧栏、Markdown 存储、自动保存（30 秒间隔 + Ctrl+S）
- **小黑屋** — 目标字数/时长设定、锁定遮罩、进度追踪
- **状态栏** — 实时字数统计、保存状态、小黑屋快捷入口
- **作品管理与欢迎页** — 新建/打开作品向导
- **竹林清风主题** — 绿色系清新配色 + 撞色点缀

### Phase 2（计划中）
- 构思空间（起名工坊、情节图谱、人设卡片、世界观画布）
- 润色空间（AI 纠错、重复检测、节奏热力图）
- AI 调度器（Ollama 本地 + 云端 API 混合）

### Phase 2b（计划中）
- 像素伙伴（数织灵感填色系统、16 种活动场景、性格标签）

### Phase 3（计划中）
- 分析空间（码字日历、时段分析、进度追踪、作品时间线、伙伴图鉴）
- 全文搜索、快照版本、导入导出

## 技术栈

| 层 | 技术 |
|----|------|
| 桌面框架 | Tauri v2 (Rust) |
| 前端 | React 18 + TypeScript 5 |
| 构建工具 | Vite 5 |
| 样式 | Tailwind CSS v4 + CSS 自定义属性 |
| 状态管理 | Zustand |
| 编辑器 | tiptap (ProseMirror) |
| 数据存储 | 本地 Markdown + JSON（零云端依赖） |

## 开始使用

```bash
# 开发模式
npm install
npm run tauri dev

# 构建 exe
npm run tauri build
```

构建产物位于 `src-tauri/target/release/bundle/`。
